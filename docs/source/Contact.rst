Contact
-------
If you find a bug or experience an unreported issue in StraKLIP, it is recommended to post a new entry in the
`Issues section <https://github.com/strampelligiovanni/StraKLIP/issues>`_ on GitHub. Feel free to propose a pull
request if you have already identified the source of the bug/issue.

If you have a global comment, inquiry about how to solve a specific task using spaceKLIP, questions or suggestions to
improve StraKLIP, please also open a new issue on GitHub and mark it with the relevant label.

If you wish to be kept informed about major StraKLIP updates and on-going/future developments, feel free to click the
'watch' button at the top of the GitHub page.
